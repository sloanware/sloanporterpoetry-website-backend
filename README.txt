Created August 2025

Welcome to the backend code of a poet's portfolio website found at
	sloanporterpoetry.com

As of 2025, the backend is being hosted on Render.
For information about the frontend, go to www.github.com/sloanware/sloanporterpoetry-website-frontend.

The backend was developed using Express in Node.js.
NPM package dependencies are Nodemailer, Validator, CORS, and express-rate-limit.
The newsletter is powered by MailerSend.
Read the full story on my sloanware website!

-Sloan

P.S. The books listed on the website are absolutely real, check them out ;)
